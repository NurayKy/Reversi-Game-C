# Reversi (Othello) Game in C

This is a terminal-based Reversi (Othello) game developed in C. 

## Features
* **User Authentication**: Sign up and sign in system with custom password hashing (`djb2` algorithm).
* **Game Save/Load System**: Users can save their current game states dynamically and resume later.
* **AI Competitor**: An automated computer opponent that calculates the best moves based on maximum flips.
* **File Operations**: User records and saved games are persistent, stored via binary file I/O operations.
